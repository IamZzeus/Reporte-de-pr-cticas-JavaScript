Ejercicios básicos de manipulación del DOM con JavaScript

El Modelo de Objetos de Documento (DOM) es una interfaz de programación para documentos web.

Funciones en JavaScript

Una función en JavaScript es similar a un procedimiento o un conjunto de instrucciones que realiza una tarea o calcula un valor, pero para que un procedimiento califique como función

Funciones de Orden Superior en JavaScript

En JavaScript, las funciones son tratados como ciudadanos de primera clase. Podemos tratar a las funciones como valores y asignarlos a otra variable, pasarlos como argumentos a otra función.

Closures en JavaScript

Un closure te da acceso al alcance de una función externa desde una función interna. En JavaScript, los closure se crean cada vez que se crea una función, en el momento de la creación de la función.
