Ejercicios básicos de manipulación del DOM con JavaScript

El Modelo de Objetos de Documento (DOM) es una interfaz de programación para documentos web.

Funciones en JavaScript

Una función en JavaScript es similar a un procedimiento o un conjunto de instrucciones que realiza una tarea o calcula un valor, pero para que un procedimiento califique como función

Arreglos en JavaScript

Los arrays son objetos similares a una lista cuyo prototipo proporciona métodos para efectuar operaciones de recorrido y de mutación.

Promesas y asincronias

La asincronía en JavaScript permite ejecutar tareas sin bloquear el hilo principal del programa. Esto es útil cuando necesitas realizar operaciones que pueden llevar tiempo, como acceder a una API o leer archivos.

Módulos en JavaScript

Los módulos de JavaScript te permiten dividir tu código en archivos más pequeños y reutilizables. Cada archivo puede contener funciones, objetos o clases, y decidir qué partes de su código exponer (exportar) a otros archivos.

Conceptos básico de POO en JavaScript

La POO se basa en cuatro principios fundamentales: abstracción, encapsulamiento, herencia y polimorfismo. En JavaScript se refiere a la habilidad de un programa para manipular y trabajar con objetos de manera genérica.
