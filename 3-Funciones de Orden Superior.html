Ejercicios Funciones de Orden Superior en JavaScript

Ejercicio 5: Función que toma otra función como parámetro

Este ejercicio se centra en la creación de una función que toma dos números y otra función como parámetros, demostrando el concepto de funciones de orden superior

Ejemplo

Se puede escojer si es una suma o una multiplicacion depende de que parametros mandemos a la funcion

Funcion

const sumar = (a, b) => a + b;

const multiplicar = (a, b) => a * b;

function ejecutarOperacion(a, b, operacion) { return operacion(a, b); }

Ejercicio 6: Función de orden superior con map

Este ejercicio muestra cómo crear una función que aplica una operación a cada elemento de un array utilizando el método map.

Ejemplo

La funcion de map pasa por cada elemento del array, de modo que multuplica por 2 cada elemento del array

Funcion

const duplicar = (x) => x * 2;

const numeros = [1, 2, 3, 4];

function aplicarOperacionAArray(arr, operacion) { return arr.map(operacion); }
