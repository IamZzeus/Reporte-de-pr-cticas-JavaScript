Ejercicios funciones basicas de manipulación de JavaScript

Ejercicio 1

Este ejercicio se centra en la creación de funciones para sumar dos números, utilizando tanto la sintaxis de función tradicional como la de función flecha, para ello puedes utilizar la consola o un documento.

Ejemplo

Donde a vale 5 y b vale 3, el resultado será 8.

Funcion tradicional

function sumar(a, b) { return a + b; }

Funcion flecha

const sumarFlecha = (a, b) => a + b;

Ejercicio 2

Manejando el caso de división por cero.

Ejemplo

Donde a vale 8 y b vale 2, el resultado será 8.

Funcion tradicional

function dividir(a, b) { if (b === 0) { return 'Error: División por cero'; } return a / b; }

Funcion flecha

const dividirFlecha = (a, b) => b === 0 ? 'Error: División por cero' : a / b;

Factorial

Se calcula el factorial de un numero

Funcion tradicional

function Fact(factorial) { for (let i = factorial - 1; i >= 1; i--) { resultado = factorial *= i; } return resultado; }

Funcion flecha

const Factor = (factorial) => { let resultado = factorial; for (let i = factorial - 1; i >= 1; i--) { resultado *= i; } return resultado; };
